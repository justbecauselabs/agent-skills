# Introduction

HeroUI Native is a modern React Native component system that focuses on fast, consistent UI building with strong theming and composable components. It pairs a utility-first styling approach (Uniwind) with a structured design token system so teams can ship consistent, accessible interfaces across platforms.

## What You Get

- **Component Library**: Prebuilt inputs, overlays, data display, and layout primitives.
- **Theming System**: Semantic tokens for color, typography, spacing, and radius with light/dark support.
- **Composable APIs**: Compound components (e.g., `Card.Header`, `Button.Label`) and slot-level styling.
- **Cross-Platform Consistency**: Shared patterns for interaction, motion, and accessibility.

## When to Use

- You want a cohesive design system that stays consistent across iOS/Android.
- You need themeable components that adapt to light/dark modes with minimal effort.
- You prefer utility-first styling with granular control per component slot.
