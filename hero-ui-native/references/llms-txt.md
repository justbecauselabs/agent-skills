# LLMs.txt

HeroUI provides LLM-friendly documentation files designed for RAG and AI tooling. Use these when you need concise, machine-readable docs.

## LLM Doc Files

- `https://v3.heroui.com/native/llms.txt` - Index of key guides, component APIs, and patterns.
- `https://v3.heroui.com/native/llms-full.txt` - Full HeroUI Native documentation corpus.
- `https://v3.heroui.com/native/llms-components.txt` - Component API docs only.
- `https://v3.heroui.com/native/llms-patterns.txt` - Common patterns, recipes, and getting-started topics.

## Doc Map (Use Case -> File)

- **Getting started**: use `llms.txt` or `llms-patterns.txt` for quick setup and provider usage.
- **Component APIs**: use `llms-components.txt` for props, usage, and examples.
- **Theming and styling**: use `llms-patterns.txt` for colors, styling, animation, and composition guidance.
- **Comprehensive search**: use `llms-full.txt` if you need everything.

## Tooling Notes

For tool-specific setup (Claude Code, Cursor, Windsurf), follow the LLMs.txt guide page in the HeroUI Native docs. You can also access the docs via the HeroUI MCP server.
